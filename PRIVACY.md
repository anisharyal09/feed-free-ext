# Privacy Policy — Feed Free - Unbiased Feed (FF - UF)

This Privacy Policy outlines how the "Feed Free - Unbiased Feed" browser extension handles data when installed on your local browser. 

### 1. Absolute Local Processing & Privacy First
We believe your data belongs to you. Feed Free is architected from the ground up to be completely decentralized and local. 
- The extension does NOT collect, track, store, or transmit your personal data, browser history, credentials, or search metadata.
- No analytics tools, third-party tracking scripts, or tracking pixels are included in this software bundle.

### 2. Browser Extensions API Usage
Feed Free reads limited structural parameters purely to execute its local user interface layout configurations:
- **Storage API:** Used exclusively to store your customized feature toggle states on your browser's local sandboxed storage.
- **Host Permissions (YouTube & Instagram):** Used strictly to identify layout DOM elements in real-time, executing CSS style blocks and MutationObservers to block algorithmic recommenders locally inside your active browser session.

### 3. Data Transmission & Security
Because zero bytes of information are collected, no user data is ever uploaded to external cloud servers, databases, or third-party networks. Your parameters are stored securely inside the browser's sandboxed storage partition.

### 4. Policy Revisions
As the product scales across milestones (such as integrating non-personalized feed alternatives in Phase 2), all configurations will continue to respect the local processing layer. Any updates to this document will be reflected directly inside this public code repository.
